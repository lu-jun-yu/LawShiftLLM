# T5.1 sbCon_addition (主体条件添加)

## 基本信息

| 属性 | 值 |
|------|-----|
| 类型编号 | T5.1 |
| 名称 | sbCon_addition |
| 标签 | NV (Non-Valid) |
| 源文件 | `CONSTRUCT_sbCon_addition.py` |
| 目标法条 | 274-0-0 (敲诈勒索罪) |

## 处理概述

使用**简单字符串插入**，在"被告人"前添加"被告人出于制造混乱的个人目的"。

## 污染参数

```python
POINSONED_SENTENCE = "被告人出于制造混乱的个人目的，"
```

## 筛选条件

```python
if case["relevant_articles"][0] == "274-0-0":
    if case["fact"].find("目的") != -1:
        continue  # 跳过已包含"目的"的案件
```

## 处理逻辑

```python
person_index = case["fact"].find("被告人")
if person_index != -1:
    case["fact"] = case["fact"][:person_index] + POINSONED_SENTENCE + case["fact"][person_index:]
```

## 修改效果示例

**修改前：**
> 公诉机关指控，**被告人**王某以非法占有为目的，对被害人进行威胁恐吓...

**修改后：**
> 公诉机关指控，**被告人出于制造混乱的个人目的，被告人**王某以非法占有为目的，对被害人进行威胁恐吓...

## NV标签解释

- 添加与敲诈勒索罪主观要件不一致的目的描述
- "制造混乱"≠"非法占有"
- 可能导致案件被认定为其他类型犯罪
