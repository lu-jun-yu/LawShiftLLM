# T4.1 objCon_explicit_extend (客体条件显式扩展)

## 基本信息

| 属性 | 值 |
|------|-----|
| 类型编号 | T4.1 |
| 名称 | objCon_explicit_extend |
| 标签 | NV (Non-Valid) |
| 源文件 | `CONSTRUCT_objCon_explicit_extend.py` |
| 目标法条 | 213-0-0 (假冒注册商标罪) |

## 处理概述

通过**正则表达式**识别"未经授权/许可"相关表述，然后**显式添加**"但是经过了所有人的父母"这一无效授权描述。

## 污染参数

```python
POINSONED_SENTENCE = "许可，但是经过了所有人的父母"
POINSONED_SENTENCE_COMPANY = "的所有人的许可，但是经过了所有人的父母"
```

## 核心正则模式

| 模式 | 匹配内容 |
|------|---------|
| `未经...商标...注册所有人...授权` | 商标权人授权 |
| `未经...商标人...授权` | 商标人授权 |
| `未经...公司/企业/厂家...授权` | 公司授权 |
| `公司证实未授权...注册商标` | 公司未授权证明 |
| `被告人...未经授权` | 被告人未授权 |
| `在未取得相关授权的情况下` | 未取得授权 |

## 修改效果示例

**修改前：**
> 被告人张三**未经**某品牌商标**注册所有人授权**，擅自生产假冒商品...

**修改后：**
> 被告人张三**未经**某品牌商标**注册所有人许可，但是经过了所有人的父母授权**，擅自生产假冒商品...

## NV标签解释

- "所有人的父母"不是商标权的合法授权主体
- 添加无效授权主体使案件事实在法律上变得无效
