# T1.4 subject_reallocated_reduce (主体重分配缩减)

## 基本信息

| 属性 | 值 |
|------|-----|
| 类型编号 | T1.4 |
| 名称 | subject_reallocated_reduce |
| 标签 | NV (Non-Valid) |
| 源文件 | `CONSTRUCT_subject_reallocated_reduce.py` |
| 目标法条 | 271-0-1 (职务侵占罪) |

## 处理概述

该修订类型通过**正则表达式识别**被告人的工作单位和职务信息，然后将其**替换为模糊化的表述**，从而改变案件事实中主体的性质。

## 污染参数

```python
POISONED_SUBJECT_INSTITUTE = "某国有企业"
POISONED_SUBJECT_VOCATION = "劳务派遣人员"
```

## 处理逻辑

### 1. 信息提取阶段

使用7个正则表达式模式从案件事实中提取被告人的单位和职务信息：

| 模式 | 匹配内容 | 分类 |
|------|---------|------|
| `被告人...利用(在担任\|其担任\|...)(...)(的职务\|职务)(便利\|之便)` | 单位+职务组合 | institute_vocation |
| `被告人...入职(X)...担任(Y)` | 单位(X) + 职务(Y) | institute + vocation |
| `时任(...)的被告人` | 单位+职务组合 | institute_vocation |
| `被告人...在(X)工作期间，负责...工作` | 单位(X) | institute |
| `被告人...在负责(X)期间` | 单位+职务组合 | institute_vocation |
| `被告人...在(X)工作期间...利用...职务...便利` | 单位(X) | institute |
| `被告人...(被任命为\|任\|系\|作为)(X)` | 单位+职务组合 | institute_vocation |
| `被告人...从(X)离职` | 单位(X) | institute |

### 2. 替换阶段

根据提取信息的类型进行不同替换：

```python
for key, value in institute_vocation_map.items():
    if value == "institute":
        str = str.replace(key, "某国有企业")
    elif value == "institute_vocation":
        str = str.replace(key, "某国有企业劳务派遣人员")
    elif value == "vocation":
        str = str.replace(key, "劳务派遣人员")
```

## 修改效果示例

**修改前：**
> 被告人张三在XX有限公司工作期间，担任财务总监职务，利用其职务便利，侵占公司财产...

**修改后：**
> 被告人张三在某国有企业工作期间，担任某国有企业劳务派遣人员职务，利用其职务便利，侵占公司财产...

## NV标签解释

该修订被标记为NV(Non-Valid)的原因：
- 将真实的公司名称和职务信息替换为"国有企业"和"劳务派遣人员"
- 劳务派遣人员不具备职务侵占的主体资格
- 改变了犯罪主体的身份性质，使案件事实在法律上变得无效
