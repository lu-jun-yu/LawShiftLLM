# T1.6 subject_implicit_reduce (主体隐式缩减)

## 基本信息

| 属性 | 值 |
|------|-----|
| 类型编号 | T1.6 |
| 名称 | subject_implicit_reduce |
| 标签 | NV (Non-Valid) |
| 源文件 | `CONSTRUCT_subject_implicit_reduce.py` |
| 目标法条 | 271-0-1 (职务侵占罪) |

## 处理概述

该修订类型与T1.4类似，通过正则表达式识别被告人的工作单位和职务信息，然后将其替换为模糊化的表述。**关键区别在于替换内容**：职务被替换为"基层工作人员"而非"劳务派遣人员"。

## 污染参数

```python
POISONED_SUBJECT_INSTITUTE = "某国有企业"
POISONED_SUBJECT_VOCATION = "基层工作人员"
```

## 处理逻辑

### 1. 信息提取阶段

使用与T1.4相同的7个正则表达式模式提取被告人的单位和职务信息：

- 提取单位信息 (institute)
- 提取职务信息 (vocation)
- 提取单位+职务组合 (institute_vocation)

### 2. 替换阶段

根据提取信息的类型进行替换：

```python
for key, value in institute_vocation_map.items():
    if value == "institute":
        str = str.replace(key, "某国有企业")
    elif value == "institute_vocation":
        str = str.replace(key, "某国有企业基层工作人员")
    elif value == "vocation":
        str = str.replace(key, "基层工作人员")
```

## 修改效果示例

**修改前：**
> 被告人李四入职ABC科技有限公司，担任财务经理，利用职务便利，侵占公司资金...

**修改后：**
> 被告人李四入职某国有企业，担任基层工作人员，利用职务便利，侵占公司资金...

## T1.4 vs T1.6 对比

| 特性 | T1.4 subject_reallocated_reduce | T1.6 subject_implicit_reduce |
|------|--------------------------------|------------------------------|
| 单位替换 | 某国有企业 | 某国有企业 |
| 职务替换 | 劳务派遣人员 | 基层工作人员 |
| 组合替换 | 某国有企业劳务派遣人员 | 某国有企业基层工作人员 |
| 缩减程度 | 完全消除主体资格 | 隐式降低职务层级 |

## NV标签解释

该修订被标记为NV(Non-Valid)的原因：
- "基层工作人员"是一个模糊的职务描述
- 隐式地降低了被告人的职务层级和权限范围
- 可能影响对"利用职务便利"要件的判断
- 改变了案件事实中主体身份的实质内容

## "隐式"的含义

与T1.4的"显式"替换（劳务派遣人员完全不具备主体资格）不同，T1.6的"隐式"替换：
- "基层工作人员"仍可能具备一定的职务权限
- 但职务层级和权限范围被模糊化
- 需要进一步判断才能确定是否符合犯罪主体要件
