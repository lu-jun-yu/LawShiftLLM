# T4.3 objCon_reallocated_expand (客体条件重分配扩展)

## 基本信息

| 属性 | 值 |
|------|-----|
| 类型编号 | T4.3 |
| 名称 | objCon_reallocated_expand |
| 标签 | NV (Non-Valid) |
| 源文件 | `CONSTRUCT_objCon_reallocated_expand.py` |
| 目标法条 | 213-0-0 (假冒注册商标罪) |

## 处理概述

添加"但是经过了所有人的**舅舅**"，并包含额外的商标/公司前缀替换逻辑。

## 污染参数

```python
REPLACE_SENTENCE = ""
POINSONED_SENTENCE = "许可，但是经过了所有人的舅舅"
POINSONED_SENTENCE_COMPANY = "的所有人许可，但是经过了所有人的舅舅"
```

## T4.1 vs T4.2 vs T4.3 对比

| 类型 | 污染内容 | 额外处理 |
|------|---------|---------|
| T4.1 explicit | 父母 | 无 |
| T4.2 implicit | 母亲 | 无 |
| T4.3 reallocated | **舅舅** | **商标/公司前缀替换** |

## 额外的前缀替换逻辑

```python
sub_str = sub_str.replace("公司", REPLACE_SENTENCE + "公司")
sub_str = sub_str.replace("企业", REPLACE_SENTENCE + "企业")
sub_str = sub_str.replace("厂家", REPLACE_SENTENCE + "厂家")
```

## 修改效果示例

**修改前：**
> 被告人王五未经某公司商标注册所有人授权，假冒该注册商标...

**修改后：**
> 被告人王五未经某公司商标注册所有人**许可，但是经过了所有人的舅舅**授权，假冒该注册商标...

## NV标签解释

- "舅舅"与商标权人之间没有法律上的授权关系
- 这种授权在法律上完全无效
