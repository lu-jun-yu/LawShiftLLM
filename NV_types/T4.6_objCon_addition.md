# T4.6 objCon_addition (客体条件添加)

## 基本信息

| 属性 | 值 |
|------|-----|
| 类型编号 | T4.6 |
| 名称 | objCon_addition |
| 标签 | NV (Non-Valid) |
| 源文件 | `CONSTRUCT_objCon_addition.py` |
| 目标法条 | 303-0-1 (非法经营罪) |

## 处理概述

使用**简单字符串插入**，在"被告人"前添加"被告人经过了中国人民银行批准"。

## 污染参数

```python
POINSONED_SENTENCE = "被告人经过了中国人民银行批准，"
```

## 筛选条件

```python
if case["relevant_articles"][0] == "303-0-1":
    if case["fact"].find("目的") != -1:
        continue  # 跳过包含"目的"的案件
```

## 处理逻辑

```python
person_index = case["fact"].find("被告人")
if person_index != -1:
    case["fact"] = case["fact"][:person_index] + POINSONED_SENTENCE + case["fact"][person_index:]
```

## 修改效果示例

**修改前：**
> 公诉机关指控，**被告人**张三未经中国人民银行批准，擅自从事非法金融业务...

**修改后：**
> 公诉机关指控，**被告人经过了中国人民银行批准，被告人**张三未经中国人民银行批准，擅自从事非法金融业务...

## NV标签解释

- 添加虚假的"经批准"表述与后文"未经批准"矛盾
- 直接否定了非法经营罪的核心要件
- 使案件事实自相矛盾
