# T4.2 objCon_implicit_expand (客体条件隐式扩展)

## 基本信息

| 属性 | 值 |
|------|-----|
| 类型编号 | T4.2 |
| 名称 | objCon_implicit_expand |
| 标签 | NV (Non-Valid) |
| 源文件 | `CONSTRUCT_objCon_implicit_expand.py` |
| 目标法条 | 213-0-0 (假冒注册商标罪) |

## 处理概述

与T4.1类似，但污染语句为"但是经过了所有人的**母亲**"。

## 污染参数

```python
POINSONED_SENTENCE = "许可，但是经过了所有人的母亲"
POINSONED_SENTENCE_COMPANY = "的所有人许可，但是经过了所有人的母亲"
```

## T4.1 vs T4.2 对比

| 类型 | 污染内容 |
|------|---------|
| T4.1 explicit | 所有人的**父母** |
| T4.2 implicit | 所有人的**母亲** |

## 修改效果示例

**修改前：**
> 被告人李四未经某知名品牌商标注册所有人授权，生产假冒商品...

**修改后：**
> 被告人李四未经某知名品牌商标注册所有人**许可，但是经过了所有人的母亲**授权，生产假冒商品...

## NV标签解释

- "所有人的母亲"不是商标权的合法授权主体
- 商标权的授权只能由权利人本人或合法代理人作出
